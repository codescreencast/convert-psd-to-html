Introduction

After you have bought the purchase, please extract zip file, you would get
  • documentation folder
  • portfolio folder
Open portfolio folder and replace images with your requirement. And you could upload all those files from portfolio folder to server. That's all for simple setup.

Customize Color

To customize something from template, you would need to install code editor or you could use default application from your operating system.
  • Notepad ( Window )
  • Gedit ( Linux )
  • TextEdit ( Mac )
But, we recommend to use one of these code editors.
  • Atom
  • Sublime Text (trial version)
  • Notepad++
  • Bracket
  • Visual Code Editor
To customize color, open style.css file from assets -> css and uncomment colors.css import file and you could add your custom color in colors.css (assets->css->colors.css) file. We also create blue custom color (#1a7def). So, you would only need to edit that color code with your one.

Customize Font

We use google fonts. Below are the fonts we use
  • Oswald ( https://fonts.google.com/specimen/Oswald?query=Oswald )
  • Source Sans Pro ( https://fonts.google.com/specimen/Source+Sans+Pro?query=Source )
You could add your custom fonts in global.css ( assets/css/global.css ) file. Don't forget to edit Link tags of google font embeded.

Images

We use demo image from unsplash website ( https://unsplash.com/ ). Image sizes we use in the template
  • Banner Image - 1519x650
  • About Us Image - 554x373

JavaScript

We use swiperjs ( https://swiperjs.com/ ) for banner and testimonial animation. You could also add your custom javascript in app.js ( assets/js/app.js )

Custom CSS

We would recommend to create your own CSS file to customize CSS and import that file in style.css. If so, you would get no conflict for future updates. But, if you don't need update anymore, you could direct edit into any CSS file depending on your requirement. Please note, we have already add most of common CSS property in common.css file ( assets/css/common.css). So, you could use those common classes in your HTML.
